# Speech Synthesis

- HTML
- CSS
- Javascript

## Usage

- Type in text input
- Choose voice for speech
- Move the rate bar to change rate of speech
- Move the pitch bar to change pitch of speech

This was hilariously fun to play around with. I also just used the starter files css.
